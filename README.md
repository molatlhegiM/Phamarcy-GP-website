# Moshate Medical Centre - Akasia, Pretoria

**Your Health, Our Priority.**

Moshate Medical Centre is a high-end, multi-disciplinary healthcare platform serving the Akasia community. This project is a production-ready, commercially sellable web application designed for modern medical facilities.

## Commercial Features

- **Mini E-Commerce Pharmacy:** A sophisticated, category-based product catalog showcasing OTC medicines and wellness products with stock-check CTAs.
- **Privacy-First Confidentiality:** Specialized discreet messaging workflows for Women's Health, using smooth modal overlays to ensure patient comfort and data dignity.
- **Service Professionalism:** Dedicated flows for Physiotherapy and Dental services, including integrated contact fallbacks to prevent user drop-off.
- **Medical Suite Management:** Integrated lead generation for healthcare professionals looking to rent consulting rooms within the facility.
- **Rule-Based Assistant:** A safe, predefined-response assistant for logistics and service guidance only.

## Design & UX

- **Premium Branding:** Consistent application of Moshate Blue (#1F7FBF) and Teal (#22C1BC) gradients for a calm, clinical, and trustworthy feel.
- **SVG-Only Iconography:** Fully replaced emojis with professional SVG icons for a corporate healthcare standard.
- **Zero-404 Architecture:** Robust hash-routing ensures compatibility with static hosting (GitHub Pages/Firebase) while eliminating broken navigation.
- **Responsive & Accessible:** Fluid layouts optimized for mobile-first patient interaction.

## Technology Stack

- **Framework:** React 19 (ES modules)
- **Typing:** TypeScript
- **Styling:** Tailwind CSS v4
- **Build Tool:** Vite
- **Deployment Ready:** Configured for GitHub Pages subpaths and Firebase Hosting.

## Regional Integration

Specifically tailored for the South African healthcare landscape:
- **Medical Aid Support:** Clear pathways for medical aid queries.
- **WhatsApp Integration:** Leveraging South Africa's primary communication channel for patient bookings.
- **Localized Content:** Specific focus on Akasia, Pretoria geography and regional health needs.
